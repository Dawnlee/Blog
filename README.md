### 个人博客网站，记录一些个人技术文章等
### 前端技术栈
* 框架 vue
* UI vuetify
### 后端技术栈
* web框架 koa2
* 数据库 mysql
### 部署
* aws ec2

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
